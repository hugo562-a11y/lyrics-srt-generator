# 歌詞 SRT 產生器

Windows 桌面工具：匯入音檔，以本機 AI 模型轉成帶時間的歌詞，補上前奏／間奏／尾奏標記，並輸出 SRT。

## 安裝

需要 Python 3.10 以上。首次開啟程式會檢查 `faster-whisper`、`soundfile`、`numpy`，缺少時會自動下載安裝；本機模型會在第一次分析時下載，之後可離線使用。

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app.py
```

Windows 使用者可直接雙擊 `run.bat`，它會建立專案專用的 `.venv` 環境並啟動程式。

## GPU（選用）

預設是「自動（GPU 優先）」。程式會在分析前確認 CUDA 12 的 `cublas64_12.dll` 與 `cudnn64_9.dll` 都能被目前 Python 行程找到；缺少時會自動從 PyPI 安裝對應 NVIDIA 執行庫並加入 DLL 搜尋路徑。其他 GPU 問題則自動回退 CPU，不會中斷分析。

若要發佈免安裝 Python 的 Windows 版本，在已完成首次啟動後執行 `build_exe.bat`；輸出於 `dist\LyricsSrtGenerator\`。請一併發佈整個資料夾，而不是只發佈 EXE。

若音檔格式無法讀取，請安裝並加入 PATH：[FFmpeg](https://ffmpeg.org/download.html)。

## 使用方式

1. 按「匯入音檔」。
2. 選擇模型與語言（中文歌曲建議 `large-v3` 或 `medium`；較快可選 `small`）。
3. 按「開始 AI 分析」。模型在本機執行，會列出每句歌詞與音樂區段。
4. 可直接雙擊表格中的開始、結束、類型或文字修改；選取一列可用工具列新增或刪除。`Ctrl+Z` / `Ctrl+Y` 可復原／重做。
5. 按「匯出 SRT」。

完成分析後，使用下方的「播放／暫停／停止」與時間滑桿試聽。播放游標進入任何歌詞、前奏、間奏或尾奏區段時，表格會自動捲動並以藍色標示該列；可在送進 Premiere 前先修正錯位。播放器使用 FFmpeg 隨附的 `ffplay`，不需要額外編譯 Python 音訊套件。

若已有正確歌詞，請先按「匯入歌詞檔」。使用 UTF-8 `.txt` 或 `.lrc`，**每句歌詞一行**；LRC 原有時間碼會忽略，改以實際音檔重抓節奏。AI 只用於找出實際演唱的時間與節奏，匯出的文字會採用歌詞檔原文，不會採用 AI 聽錯的字。

### 提升對齊準確度

- 保持勾選「精準逐字對齊」：Whisper 會產生逐字時間錨點，再把正確歌詞逐行套上，通常比整句時間範圍精準。
- 伴奏較大聲、合唱或拖音明顯時，勾選「先分離人聲」：程式會按需下載 Demucs 與其本機模型，先從音檔抽出人聲再對齊。第一次較久，之後模型會快取。
- 人聲分離是選用功能；若其相依套件無法在目前 Python 版本安裝，關閉勾選仍可使用逐字對齊。

## 標記規則

- 第一個歌詞前：`[前奏]`
- 歌詞句與句間的音樂空檔：`[間奏]`
- 最後一句後：`[尾奏]`
- 較短的呼吸／停頓不會產生 SRT 項目；可在介面調整「最短音樂段」。

SRT 是 `HH:MM:SS,mmm` 規格，因此第一筆從 `00:00:00,000`（等同所要求的 `00:00:00:00` 起點）開始。
