# 2026-07-15 開發進度

## 修正項目

### 預覽窗無限變寬
- **問題**：使用 `tk.Label` + `image=` 後，每次設定圖片會觸發 Configure 事件，導致 widget 不斷自動放大。
- **解法**：將 `tk.Label` 改為 `tk.Canvas` 繪製預覽圖，widget 大小由 grid 佈局決定，不再受圖片影響。
- **相關提交**：`523b458`

### 預覽窗尺寸無法適配面板
- **問題**：預覽窗使用固定 480×270 渲染尺寸，文字底部被裁切。
- **解法**：`_preview_dimensions()` 改為讀取 Canvas 實際 `winfo_width/height`，渲染尺寸自動適配面板大小。
- **相關提交**：`f5cd446`

## 新增功能

### 逐字動畫效果重寫
- **目標**：12 種動畫效果各有獨特的逐字動態表現，不再全部像打字機。
- **做法**：為每種動畫加入 `karaoke_reveal`（依序浮現）搭配各自獨立的視覺效果。
- **各動畫效果**：

| 動畫 | 效果描述 |
|------|---------|
| 逐字點亮 | 字依序亮起金黃光暈，scale 微幅脈衝 |
| 彈跳聚焦 | 字從上方逐字掉落並彈跳 |
| 滑入淡出 | 整行一起滑入（不逐字），純 enter/leave 動畫 |
| 電影柔和 | 整行柔和淡入（不逐字），低光暈 |
| 暴風雨 | 字逐字閃爍震動，搖晃幅度隨 active 增大 |
| 脈衝擴散 | 字逐個產生擴散環 |
| 水波震盪 | 字逐字上下左右波動 |
| 雷射掃過 | 雷射線逐字掃過，命中字發紅光+加粗描邊 |
| 氣泡彈出 | 字逐個從下方彈出，squeeze 變形 |
| 殘影拖曳 | 字逐字帶殘影滑入 |
| 閃爍霓虹 | 字逐個霓虹閃爍，紅/綠/藍隨機色 |
| 粒子爆破 | 字逐個散開再聚攏 |
- **相關提交**：`730d450`、`9199147`

## 技術備註
- `karaoke_reveal = _smooth(max(0, min(1, (local - karaoke * char_progress) / (.15 / speed))))`
  - `karaoke`：字數越多時越長，最短 0.55s、最長為 duration * 70%
  - `char_progress`：0.0（第一字）→ 1.0（最後一字）
  - 搭配 `_smooth()` 三次方緩動，每字依序浮現
- 滑入淡出/電影柔和不使用 `karaoke_reveal`，維持整行 enter/leave 效果
